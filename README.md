# Credit-Loan-Management-System
Credit Risk Assessment &amp; Loan Management System  Developed a machine learning-based system to evaluate applicant creditworthiness and automate loan approval decisions. Performed data preprocessing, feature engineering, and predictive modeling using classification algorithms. Built a dashboard for risk visualization and loan application analysis. 
# Credit Risk Assessment & Loan Management System

## Overview
An AI-powered system that analyzes applicant financial information to assess credit risk and assist in loan approval decisions.

## Features
- Credit risk prediction
- Loan approval classification
- Data preprocessing and feature engineering
- Interactive dashboard
- Model performance evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Streamlit (if used)

## Dataset
The dataset contains applicant information such as:
- Income
- Loan Amount
- Credit History
- Employment Status
- Loan Status

## Model
Machine Learning models used:
- Logistic Regression
- Random Forest
- Decision Tree

## Results
Achieved high accuracy in predicting loan approval and identifying risky applicants.

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   streamlit run app.py
